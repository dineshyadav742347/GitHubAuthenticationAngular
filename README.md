Description:
This code is sample code of Github authrization. It is using passport Oauth.


Tech Stack:
Angular 4
Expressjs
Nodejs
Passport (GitHub Oauth)
Mongo


Steps to run code:-
1) Go to server/config/ folder and enter your GitHub client secret and client id into appConfig.js file.
2) Go to client folder and run "ng build" command on command prompt. This process will create bundle of frontend code.
3) Go to server folder and run "npm start" command on command prompt. it will start your server.
4) Go to your browser and enter "http://localhost:3000/".
5) Your code is start now.


Note:- This exercise is only to implement Github authentication. If you want good UI, you can customize your ui according to your requirments.
